# Evidencia Visual – Capturas de Pantalla

Agregar aquí las capturas de pantalla de los reportes HTML de JMeter.

## Capturas requeridas

| Archivo | Cómo obtenerla |
|---------|---------------|
| `baseline_statistics.png` | Abrir `perf/results/baseline_report/index.html` → sección **Statistics** → captura completa de la tabla |
| `baseline_response_time.png` | Mismo reporte → sección **Response Time Over Time** o **Response Time Percentiles** |
| `load_statistics.png` | Abrir `perf/results/load_report/index.html` → sección **Statistics** |
| `load_active_threads.png` | Mismo reporte → sección **Active Threads Over Time** |

## Cómo abrir los reportes

```
# Windows
start perf\results\baseline_report\index.html
start perf\results\load_report\index.html
```
